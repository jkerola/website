How did you make it?

This website was built with Flask.
Content made with flask-blogging is stored in an SQLite database using SQLAlchemy. The frontend is bog-standard HTML and CSS, with styles and minimal javascript from Bootstrap. Served from a DigitalOcean droplet straight to your browser.
